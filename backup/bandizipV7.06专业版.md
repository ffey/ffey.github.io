# 目录

# 1.bandizip的安装
1. 首先双击安装程序
![a](https://img.picui.cn/free/2024/06/24/6679327bb3126.png)
2. 在弹出的页面里点击否
![b](https://img.picui.cn/free/2024/06/24/66793418b1251.png)
3. 点击同意并安装
# 2.激活bandizip
1. 打开bandizip的安装目录
2. 把文件夹的注册机给拖进去
![c](https://img.picui.cn/free/2024/06/24/6679362506af4.png)
3. 打开注册机`Ctrl`+`C`复制专业版离线密钥并点击应用
![e](https://img.picui.cn/free/2024/06/24/667937681193a.png)
4. 出现运行完毕即可
![e](https://img.picui.cn/free/2024/06/24/6679376cb4b87.png)
5. 打开bandizip
6. 点左上角的钥匙图标
![f](https://img.picui.cn/free/2024/06/24/6679390420fce.png)
这里邮箱地址随便填
![j](https://img.picui.cn/free/2024/06/24/66793906ea5ec.png)
出现这个就说明激活成功了|
